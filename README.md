# Natural Language to SQL Project

This project is a modular Python and SQLite system for loading structured CSV data into a database and supporting safe query workflows.

## Current Progress
- Project scaffold created
- SQLite connection helper implemented
- Schema manager implemented
- Unit tests added for DB connection and schema logic

- Schema normalization added
- Schema comparison logic added
- Decision flow added for create vs append vs conflict

## Tech Stack
- Python
- SQLite
- pandas
- pytest
- GitHub Actions
